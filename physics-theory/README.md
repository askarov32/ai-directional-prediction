# Physics Theory

Эта директория содержит отдельные русскоязычные материалы по физической постановке задачи:

- какие физические данные используются в проекте;
- какие физические параметры подаются в модель;
- какие формулы лежат в основе термоупругой постановки;
- какие формулы реально используются в текущем MVP-коде `PINN` и в постобработке предсказаний.

## Важное замечание

Здесь специально разделены два уровня:

1. **Физическая теория задачи**
   Это классическая термоупругость и волновая механика.

2. **Текущая MVP-реализация**
   Это практический гибрид:
   - часть физики заложена через PINN loss;
   - часть через физически осмысленные признаки;
   - часть через постобработку предсказаний в `pinn-service`.

То есть текущий сервис не является полным численным PDE-решателем в реальном времени. Он использует:

- обучаемую нейросеть;
- физические ограничения;
- инженерные формулы и эвристики для формирования финального предсказания.
